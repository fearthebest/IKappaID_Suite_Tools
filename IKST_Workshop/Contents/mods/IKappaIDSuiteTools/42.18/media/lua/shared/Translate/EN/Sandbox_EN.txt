Sandbox_IKappaIDSuiteTools = IKappaID Suite Tools

Sandbox_IKappaIDSuiteTools_EnableMod = Turn on the mod

Sandbox_IKappaIDSuiteTools_VehicleListRadius = How far to search for cars (Old - use Vehicles addon instead)

Sandbox_IKappaIDSuiteTools_VehicleNearRadius = How close you must be to claim the nearest car

Sandbox_IKappaIDSuiteTools_ClaimPlayerSelfService = Let players claim cars and houses for themselves

Sandbox_IKappaIDSuiteTools_MaxVehicleClaims = How many cars one player can own (0 means no limit)

Sandbox_IKappaIDSuiteTools_MaxSafehouseClaims = How many houses one player can own (0 means no limit)

Sandbox_IKappaIDSuiteTools_ClaimDurationDays = How many days a claim lasts (0 means it never ends)

Sandbox_IKappaIDSuiteTools_ClaimAdminBypass = Admins can use any claimed car

Sandbox_IKappaIDSuiteTools_ClaimWhitelistOnly = Only the owner and named players on the whitelist can access claims (everyone else is blocked)

Sandbox_IKappaIDSuiteTools_ClaimAllowNamedPlayers = Let owners add specific players to a whitelist with custom permissions

Sandbox_IKappaIDSuiteTools_ClaimMaxNamedPlayers = Max players on a whitelist per claim (0 means no limit)

Sandbox_IKappaIDSuiteTools_ClaimOwnersGrantExtra = Owners can give whitelisted players more access than the guest settings below allow

Sandbox_IKappaIDSuiteTools_ClaimOwnersEditGroups = Let owners change Everyone, Members, Safehouse mates, and Faction tabs on their claims

Sandbox_IKappaIDSuiteTools_ClaimVehicleGuestEnter = Let strangers get inside claimed cars (ignored when whitelist-only mode is on)

Sandbox_IKappaIDSuiteTools_ClaimVehicleGuestDrive = Let strangers drive claimed cars (ignored when whitelist-only mode is on)

Sandbox_IKappaIDSuiteTools_ClaimVehicleGuestLoot = Let strangers take items from claimed cars (ignored when whitelist-only mode is on)

Sandbox_IKappaIDSuiteTools_ClaimVehicleGuestDoors = Let strangers open doors and windows on claimed cars

Sandbox_IKappaIDSuiteTools_ClaimVehicleGuestRefuel = Let strangers refuel claimed cars

Sandbox_IKappaIDSuiteTools_ClaimVehicleMateEnter = Let safehouse mates get inside the owner's claimed cars

Sandbox_IKappaIDSuiteTools_ClaimVehicleMateDrive = Let safehouse mates drive the owner's claimed cars

Sandbox_IKappaIDSuiteTools_ClaimVehicleMateLoot = Let safehouse mates take items from the owner's claimed cars

Sandbox_IKappaIDSuiteTools_ClaimSafehouseGuestBuild = Let strangers build inside claimed houses (ignored when whitelist-only mode is on)

Sandbox_IKappaIDSuiteTools_ClaimSafehouseGuestDestroy = Let strangers break things inside claimed houses (ignored when whitelist-only mode is on)

Sandbox_IKappaIDSuiteTools_ClaimSafehouseGuestLoot = Let strangers take items from claimed houses (ignored when whitelist-only mode is on)

Sandbox_IKappaIDSuiteTools_ClaimSafehouseGuestDoors = Let strangers open doors in claimed houses

Sandbox_IKappaIDSuiteTools_ClaimSafehouseMemberBuild = Let safehouse members build inside claimed houses

Sandbox_IKappaIDSuiteTools_ClaimSafehouseMemberDestroy = Let safehouse members break things inside claimed houses

Sandbox_IKappaIDSuiteTools_ClaimSafehouseMemberLoot = Let safehouse members take items from claimed houses

Sandbox_IKappaIDSuiteTools_RecoveryJournalEnabled = Turn on the skill book (save and get back your skills)
